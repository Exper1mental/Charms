For Windows 8.0 an above.

This program allows Windows Toast Notifications to be shown from the Desktop.
On running a shortcut will be created with the same AppUserModelID:
  <http://msdn.microsoft.com/en-us/library/windows/desktop/dd378459>
The shortcut is creatd in the current Users Start Menu location:
  C:\Users\%username%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\
The shortcut will be named whatever the exe file is called so multiple can be created.

Can be run from another application or create a shortcut and add parameters:
  Three parameters are accepted
    1. Title (Shown with Bold Font)
    2. Line 1
    3. Line 2
  Additionally if ToastImage.png is in the same folder this Image will be shown.

   ******************************************************
   *            *                                       *
   *            *     TITLE                             *
   *   IMAGE    *     Line 1                            *
   *            *     Line 2                            *
   *            *                                  Icon *
   ******************************************************

Example: L2ToastNotification.exe "Paul's Programs" "http://www.l2adv.com/" "L2 ADVanced"
Note: parameters should be enclosed in quotes.
      Each line can accomodate about 27 characters.  If longer ... will be shown

   ******************************************************
   *            *                                       *
   *            *     Paul's Programs                   *
   *   IMAGE    *     http://www.l2adv.com/             *
   *            *     L2 ADVanced                       *
   *            *                                  Icon *
   ******************************************************
